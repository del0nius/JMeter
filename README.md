## Работа с Apache JMeter — инструментом для проведения нагрузочного тестирования. ##

* В папке Gaponenko_HW_1_JMeter выполнена домашняя работа №1 (в .jmx файле находится тестовый сценарий с запросами при нагрузке в 250 потоков - 50 пользователей по 5 запросов от каждого).

* В папке Gaponenko_HW_2_JMeter выполнена домашняя работа №2 (используются инструменты: JSON Extractor, BeanShell Assertion для передачи значений параметров между запросами, Response Assertion для проверки статус кодов запросов и JSON Assertion для проверки соответствия между отправленными и полученными значениями в запросах).
